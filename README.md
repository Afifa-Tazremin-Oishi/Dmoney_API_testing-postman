# Dmoney_API_testing-postman

## Project Summary
This project has automate user API from postman using newman

## Technology Used:
- Postman
- Newman

## Prerequisites:
- NodeJS

## How to run:
- ``` git clone ```
- ``` npm i ```
- ``` node .\report.js ```

A report will be generated in your project source root
